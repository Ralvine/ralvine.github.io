# 火车开往破晓

!!! abstract ""
    **「昔人已乘黄鹤去，此地空余黄鹤楼。」五一前夕，一个念头在脑海中迸发：之前走过东西南北的不同省份，却唯独忘记了华中这片土地。于是，这趟行程就此展开。**

先前对武汉的了解还只是停留在疫情初期的报道和「下一站，茶山刘」的寥寥几句歌词，黄鹤楼和它经久不衰的诗篇确有让人一探究竟的兴趣，但更令我心生向往的还是荆楚大地的当代景观，它似乎没有沿海的繁华或是川渝的喧嚣，而恰恰是这种内敛的城市印象构成了其近代风云变幻的厚重历史篇章的一道浓雾。

## 前期准备

刚开始规划武汉行程时，我们期望能错开五一假期的游客高峰，因此决定在27日晚上启程，整个旅程时间压缩在三天左右，总体来看行程还是比较紧张的。起初南昌也被安排在计划之列，后来因为时间原因遂弃。

和高中同学完成初步的规划后，我们又在朵朵上攒到了两位朋友。说起来，自己一直觉得对于一周时段内、跨省的「中途」旅行而言，两到四个人是相对舒适的组队规模，在打车、就餐上能够兼顾便利性和乐趣，也能节省一定的预算。

!!! success "实际行程"
    - **04.27** 杭州-汉口站-光谷步行街
    - **04.28** 湖北省博物馆-武汉大学-东湖-万松园
    - **04.29** 昙华林-粮道街-司门口-黄鹤楼-长江大桥-江汉路
    - **04.30** 南湖环线-杭州

我们的实际行程同飞书上的安排基本一致，但一路游览下来后难免感到有些走马观花，单日数万步的运动量在人数较多的情况下也会遇到各样的不便，不过好在整趟旅程的体验还是满意的，街道、景区热闹但不拥堵，博物馆、黄鹤楼都比较容易预约、购票，尽管后者的排队体验有些糟糕。印象比较深刻的是傍晚骑行穿过长江大桥的过程，能感受到华中地区的历史气息，远眺黄鹤楼、俯瞰沿线铁路也是颇有趣味。
